◆ iteraciones()

void iteraciones ( )

interaciones() encuentra el pivote en la tabla, hace las iteraciones y crea las nuevas tablas y encuentra la solucion optima.

Esta funcion recorre la tabla inicial hasta encuentrar el pivote de las restricciones, realiza las operaciones a las demas restricciones para formar una nueva tabla o en caso de haber encontrado la solucion optima imprimira la tabla final.


Parameters
maximoGuarda el valor absoluto de las restricciones
columnaSe posiciona en la columna del pivote
tempGuarda temporalmente el valor de la restriccion en las posicion de rest y j multiplicada por -1
countGuarda un indice hasta ser igual a rest
auxVariable auxiliar para encontrar el valor menor de las restricciones, y ayuda a la creacion de la nueva tabla, tomando el valor del pivote, para dividir a las demas restricciones.
menorGuarda el valor menor de las restricciones
cuentaSirve como contador de las restricciones mas las variables
